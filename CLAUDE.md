# 금융 시황 조회

## 프로젝트 설명
일간, 주간, 월간 금융 시황을 볼 수 있는 웹페이지입니다.

## 사용 기술
- **HTML**: 웹페이지의 구조와 콘텐츠
- **CSS**: 스타일링 및 레이아웃
- **JavaScript**: 동적 기능 및 데이터 처리

## 파일 구조
```
260207/
├── index.html          # 메인 HTML 파일 (웹페이지 구조)
├── style.css           # 스타일시트 파일 (디자인 및 레이아웃)
├── script.js           # JavaScript 파일 (기능 구현)
└── CLAUDE.md           # 프로젝트 문서 (이 파일)
```

## 실행 방법
1. 프로젝트 폴더에서 `index.html` 파일을 찾습니다
2. `index.html` 파일을 더블클릭하거나 브라우저로 드래그하여 엽니다
3. 또는 브라우저에서 파일 > 열기 메뉴를 통해 `index.html`을 선택합니다

## 코딩 규칙

### 1. 한국어 주석 사용
모든 주석은 한국어로 작성하여 이해하기 쉽게 합니다.
```javascript
// 좋은 예
const totalPrice = price * quantity; // 총 가격 계산

// 나쁜 예
const totalPrice = price * quantity; // Calculate total price
```

### 2. 초보자가 이해하기 쉬운 코드
- 명확하고 의미있는 변수명 사용
- 복잡한 로직은 단계별로 분리
- 간단하고 직관적인 구조 유지

```javascript
// 좋은 예: 의미있는 변수명
let userName = "홍길동";
let userAge = 25;

// 나쁜 예: 불명확한 변수명
let x = "홍길동";
let y = 25;
```

### 3. 코드마다 설명 주석 달기
각 주요 기능과 로직에 설명 주석을 추가합니다.

```javascript
// 사용자 정보를 가져오는 함수
function getUserInfo() {
    // 로컬 스토리지에서 사용자 이름 가져오기
    const name = localStorage.getItem('userName');

    // 사용자 이름이 없으면 기본값 설정
    if (!name) {
        return '게스트';
    }

    // 사용자 이름 반환
    return name;
}
```

### 4. 코드 작성 시 주의사항
- 들여쓰기는 공백 4칸 또는 탭 1개로 통일
- 함수와 변수는 용도를 명확히 알 수 있는 이름 사용
- 중복 코드는 함수로 분리하여 재사용
- 주석은 "왜"를 설명하고, 코드는 "무엇"을 보여줌
